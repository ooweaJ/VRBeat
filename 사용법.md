  사용법
  cd Tools
  python generate_beatmap.py "C:\내노래.mp3" --title "곡 제목" --artist
  "아티스트"
  → Assets/StreamingAssets/Songs/<곡>/ 에 오디오+차트 생성 → Unity
  새로고침하면 SongSelect에 곡이 뜹니다.

  동작 핵심: songOffset=0, beat = 시각(초) × bpm / 60. Conductor가 beat를
  시간으로 되돌리면 정확히 그 오디오 지점에 노트가 도달합니다.
  Easy=비트마다, Normal=온셋마다(1/2박 양자화), 색은 좌우 교대.

  알아둘 한계
  - 게임이 고정 BPM 격자라 변박 곡은 뒤로 갈수록 어긋날 수 있음 → 단일 BPM
   곡에 최적.
  - BPM이 절반/2배로 잡히면 --bpm 128로 강제.
  - 전체적으로 박자가 밀리면 info.json의 songOffset을 ±0.02 단위로
  미세조정.
  - lane/방향/색은 음악적 분석이 아니라 패턴 생성(플레이감 위주). 더
  정교하게 하려면 madmom이나 규칙 기반으로 확장 가능.
